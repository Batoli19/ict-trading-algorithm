This folder contains the Backtesting Engine. It is used to simulate the bot's trading strategy on historical data without risking real money.

To run a backtest:
You can use one of the existing 'run_pass*.py' files here to test different variations or settings of the strategy. It will read historical data, simulate the trades, and save the performance results to a CSV file in the 03_BACKTEST_RESULTS folder. This is how we prove the strategy is profitable before taking it live.
